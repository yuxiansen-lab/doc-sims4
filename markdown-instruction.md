# Markdown 快速参考手册

这是一份专门为本站编写者准备的 Markdown 语法指南，涵盖了基础语法和 VitePress 特有的增强功能。

## 1. 基础排版

### 标题
使用 `#` 号来定义标题等级：
`#` 一级标题 (页面大标题)
`##` 二级标题 (章节标题)
`###` 三级标题 (小节标题)

### 文本装饰
- **加粗**: `**这是加粗文字**`
- *斜体*: `*这是斜体文字*`
- ~~删除线~~: `~~这是删除线~~`
- `行内代码`: \`这是行内代码\`

---

## 2. 列表与引用

### 无序列表
使用 `-` 或 `*`：
- MOD 文件夹
- Tray 文件夹
- Saves 文件夹

### 有序列表
使用数字：
1. 下载文件
2. 解压压缩包
3. 放入文件夹

### 引用
使用 `>`：
> **注意**：脚本 MOD（.ts4script）通常不能放超过一级子文件夹。

---

## 3. 链接与媒体

### 链接
语法：`[显示文字](链接地址)`
示例：[前往 MCCC 官网](https://deaderpool-mccc.com/)

### 图片
语法：`![图片描述](图片路径)`
示例：`![我的截图](/sims4-screen.png)`
*注：图片建议放在 `.vitepress/public` 目录下。*

## 5. VitePress 特色容器 (推荐使用)
VitePress 提供了一些非常精美的容器，适合用来写提示或警告：

::: info 信息
这是一个普通的蓝框信息提示。
:::

::: tip 技巧
这是一个绿色的技巧或建议框。
:::

::: warning 警告
这是一个黄色的警告框，用来提醒注意风险。
:::

::: danger 危险
这是一个红色的危险框，通常用来提示“会导致存档损坏”等操作。
:::
