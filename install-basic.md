# 入门：安装与路径

想要让《模拟人生 4》拥有无限可能，正确安装 MOD 是第一步。本教程将带你避开 90% 新手都会遇到的坑。

---

## 1. 核心文件夹：Mods 在哪里？

首先，你必须找到存放 MOD 的“圣地”。请注意：**它不在你的游戏安装目录里！**

### 📂 路径定位
请打开你的文件资源管理器，复制并粘贴以下路径：

`Documents\Electronic Arts\The Sims 4\Mods`

或者手动点击：
**此电脑** > **文档** > **Electronic Arts** > **The Sims 4** > **Mods**

::: warning 重要提醒
如果你在 `The Sims 4` 文件夹里没看到 `Mods` 文件夹，请先启动一次游戏。系统会在你第一次进入主菜单时自动创建它。
:::

---

## 2. 开启游戏内的“模组开关”

放好文件后，必须在游戏设置里手动“批准”模组运行。

1. 进入游戏主界面，点击 **游戏选项 > 其他**。
2. 勾选以下两项：
    - [x] **启用自定义内容与模组**
    - [x] **已允许脚本模组** (这一项不勾，WW等功能包将无法运行)
3. 点击 **应用更改** 并 **彻底退出并重新启动游戏**。

---

## 3. 文件存放规则 (新手必看)

下载的 MOD 解压后，通常会看到两种格式的文件，它们的规则**完全不同**：

### 🔵 .package 文件 (家具/衣服/皮肤)
- **存放位置**：直接丢进 `Mods` 文件夹。
- **层级要求**：可以套娃。例如 `Mods/衣服/作者名/夏季款`，游戏依然能识别。

### 🔴 .ts4script 文件 (脚本模组/功能包)
- **存放位置**：例如 WW (绅士包)、MCCC、UI 签到等。
- **层级限制**：**极其严格！** 只能放在 `Mods` 根目录，或者**最多嵌套一层子文件夹**。

::: danger 致命错误演示
如果你把脚本文件放得太深，它们将**彻底失效**：
- ✅ **正确**：`Mods/WickedWhims/WickedWhims.ts4script` (1层)
- ❌ **错误**：`Mods/功能类/绅士包/WickedWhims.ts4script` (2层，无法识别)
:::

---

## 4. 日常维护习惯

为了保证游戏不卡顿、不报错，建议养成以下习惯：

1. **清理缓存**：每次安装或删除 MOD 后，删除 `The Sims 4` 目录下的 `localthumbcache.package` 文件。
2. **分类管理**：不要把几千个文件直接堆在 `Mods` 根目录，按功能建立文件夹（例如：`/功能`、`/家具`、`/外观`）。
3. **文件夹命名**：尽量使用英文或简单的拼音，避免使用特殊符号。
